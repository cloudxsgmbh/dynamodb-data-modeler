# DynamoDB Data Modeler

A local React + Vite SPA for visually modeling Amazon DynamoDB table schemas. Create virtual tables, define partition/sort keys, add GSIs, manage item types, run queries, and import/export NoSQL Workbench and OneTable schemas — all in the browser, no AWS account needed.
